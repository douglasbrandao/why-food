## Requisitos

* [Node](https://nodejs.org/en/)
* [Yarn](https://classic.yarnpkg.com/en/docs/install)

### Rodando

Instale as dependências

```yarn```

Execute o projeto

```yarn start```

Para acessar http://localhost:3000
